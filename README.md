A small project with react-hook-form. I trained here zod validation based on input choose using zod union and communication with external api using axios and react-query. 

### **Node version:**
* 14.17.3

### **Npm version:**
* 8.11.0

### **Setup**
To run this project

Docker:

`docker-compose up `

How to run project locally:

`npm install`

`npm start`
